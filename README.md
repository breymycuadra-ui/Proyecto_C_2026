# Proyecto C

## Descripción 
Este proyecto es un conversor de moneda básico, que realiza cambios entre dolares, quetzales y euros.

### Pre requisitos 
1. Debemos instalar el compilador de C utilizando: Sudo apt install build-essential
2. Comando para ver la versión del compilador de C: gcc --version

#### Pasos para ejecutar el proyecto
Para compilar el programa escribimos en la terminal de vs code: ./conversor
